## 문제 풀이 

DP는 작은 연산을 memoization 하기 위해 d 배열을 사용한다. 

d 배열은 경우의 수를 담고 있다. 

d[1]은 타일이 2*1 일 때 배치할 경우의 수를 의미한다. 

d[2]는 타일이 2*2 일 때 배치할 경우의 수이다. 

직접 손으로 계산해 봤을 때는 이렇게 나왔다. 
`d[1] = 1 
d[2] = 2
d[3] = 3
d[4] = 5
d[5]= 8`

직접 해봤을 때만 보고 `d[i] = d[i-1] + d[i-2]`를 도출하는게 아니라, 

타일을 배치할 때 결국 맨 끝은  새로 한 개일 경우(ㅣ) or 가로 두 개일 경우 ( = )  이렇게 두 가지 경우로 나뉜다. 

만약 세로 한 개 ( | ) 의 블럭을 배치할 경우 → 2*(n-1) 이 된다. (n은 타월의 가로의 길이이다.)

가로 두 개 ( = ) 의 블럭을 배치할 경우는 → 2*(n-2) 가 된다. 

따라서 d[i] = d[i-1] + d[i-2] 이 식을 도출할 수 있다.
